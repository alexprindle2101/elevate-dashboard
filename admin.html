Chat Notifications

Automatically post sale notifications to your team's group chat when reps submit orders.

How to set up Discord webhook:
  1. Open your Discord server
  2. Right-click the channel you want notifications in
  3. Click Edit Channel → Integrations
  4. Click Webhooks → New Webhook
  5. Name it (e.g. "Sales Bot"), then click Copy Webhook URL
  6. Paste the URL above
How to set up GroupMe bot:
  1. Go to dev.groupme.com/bots and sign in
  2. Click Create Bot
  3. Select the group chat you want notifications in
  4. Give it a name (e.g. "Sales Bot") and optionally an avatar URL
  5. Click Submit
  6. Copy the Bot ID and paste it above
