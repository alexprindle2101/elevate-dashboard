Adding a New Automated Channel
How to set up a new channel for automatic management:
  1. Create the channel in Slack (public or private)
  2. Invite the bot: type /invite @Aptel Channel Auditor in the channel
  3. Come back here and add the mapping below (Department or Role)
  4. The system will automatically add/remove people when their user group changes

Note: The bot can only manage channels it has been invited to. For private channels, you must manually invite the bot first.
